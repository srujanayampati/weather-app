## 📖 About The Project

Ever wanted to know the weather in any city instantly? This app does exactly that!
Search for any city and get live temperature, humidity, wind speed and a 5-day
forecast — all powered by the OpenWeatherMap API. Built this to learn API
integration and async JavaScript! ⛅

---

## ✨ Features

- 🔍 Search weather for any city in the world
- 🌡️ Shows temperature in Celsius / Fahrenheit
- 💧 Humidity and wind speed details
- 📅 5-day weather forecast
- 🎨 Dynamic background changes based on weather
- 📱 Fully responsive on all screen sizes

## 🚀 Getting Started

### Prerequisites
- A free API key from [OpenWeatherMap](https://openweathermap.org/api)

### Installation

```bash
# Clone the repository
git clone https://github.com/srujanayampati/weather-app.git

# Go into the folder
cd weather-app

# Open in browser
open index.html
```

### Setup API Key
1. Go to [openweathermap.org](https://openweathermap.org) and create a free account
2. Copy your API key
3. Open `script.js` and replace:
```js
const API_KEY = "your_api_key_here";
```
